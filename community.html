Firebase Setup Guide

Follow these steps to connect your Firebase project and enable real-time features.

1
Create a Firebase project

Go to console.firebase.google.com and click "Add project".

2
Enable Authentication

In your project → Build → Authentication → Sign-in method → Enable Email/Password.

3
Create Firestore Database

Build → Firestore Database → Create database → Start in test mode.

4
Enable Realtime Database

Build → Realtime Database → Create database → Start in test mode.

5
Enable Storage

Build → Storage → Get started → Start in test mode.

6
Copy your config

Project Settings → Your apps → Web app → Copy the firebaseConfig object and replace the placeholder values in community.html.
